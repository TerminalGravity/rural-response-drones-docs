## Phase 0 — Research OS

Status: in progress.

- launch this docs site
- collect grant sources
- collect competitor/vendor data
- write viability verdict
- define first offer
- draft concept note

## Phase 1 — Validation

Goal: determine whether agencies want help.

Actions:

- schedule 10 stakeholder conversations
- document current drone capabilities
- identify unfunded needs
- map grant deadlines
- collect letters/emails of interest
- choose first pilot partner

## Phase 2 — Pilot Proposal

Goal: produce one fundable package.

Outputs:

- 2-page concept note
- budget
- equipment spec
- grant narrative
- implementation schedule
- risk/insurance plan
- FAA/SOP checklist
- partner roles

## Phase 3 — Funded Implementation

Goal: win grant/foundation/local support and implement agency-owned capability.

Outputs:

- procurement support
- training calendar
- exercise plan
- monthly readiness process
- reporting dashboard
- maintenance plan

## Phase 4 — Regional Expansion

Goal: repeat in other rural recreation markets.

Targets:

- Bullhead City
- Kingman
- Yavapai / Prescott region
- Coconino-type SAR agencies
- Nevada/California river and desert communities
- tribal public safety partners
