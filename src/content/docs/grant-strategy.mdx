# Grant Strategy

## Core principle

The grant applicant should usually be the agency, not the startup.

The startup should provide grant strategy, implementation, training, maintenance, and readiness services.

## Strongest grant lanes

### FEMA Homeland Security Grant Program / SHSP

Best for interoperable communications, incident command, UAS as disaster/SAR situational awareness, and multi-agency preparedness.

- FEMA: https://www.fema.gov/grants/preparedness/homeland-security
- Arizona: https://azdohs.gov/shsgp

### FEMA Assistance to Firefighters Grant

Best if fire/EMS leads the application and the drone capability is tied to responder safety, rescue, hazardous scene reconnaissance, or fireground overwatch.

- https://www.fema.gov/firegrants

### DOJ Byrne JAG

Best if sheriff/police lead the application for public-safety technology, missing-person response, evidence search, officer safety, or incident response.

- https://bja.ojp.gov/program/jag/overview

### USDA Community Facilities

Best for smaller rural communities, tribes, districts, or public bodies seeking emergency-response equipment or essential public-safety infrastructure.

- https://www.rd.usda.gov/programs-services/community-facilities/community-facilities-direct-loan-grant-program

### Arizona State Parks OHV Fund

Best local-fit source for desert/OHV rescue, trail safety, off-road incident prevention, and remote access capability.

- https://azstateparks.com/off-highway-vehicle-fund-grants

### State Lake Improvement Fund

Best for Lake Havasu marine rescue, boating safety, navigation aids, rescue access, and water-safety prevention.

- https://azstateparks.com/state-lake-improvement-fund-grants

### Firehouse Subs Public Safety Foundation

Best private equipment grant for AEDs, rescue gear, radios, public safety equipment, and possibly drone-adjacent rescue capability.

- https://firehousesubsfoundation.org/

### ARDC Grants

Best for rural emergency communications, repeaters, radio/data systems, SAR comms, and resilient communications projects.

- https://www.ardc.net/apply/

## Funding package pattern

| Project | Lead applicant | Best funding sources |
|---|---|---|
| Desert/OHV SAR drone kit | Sheriff SAR / county / nonprofit | OHV Fund, Firehouse Subs, Yamaha/Polaris, Walmart local |
| Lake rescue drone support | City/county/marine patrol/fire | SLIF, Firehouse Subs, BoatUS/Sea Tow, local sponsors |
| Comms + command kit | County EM / sheriff / ham nonprofit | SHSP, ARDC, 9-1-1 program, EMPG |
| Fire/EMS overwatch | Fire/EMS agency | AFG, SAFER, Firehouse Subs |
| Rural response program | County / district / tribe | USDA Community Facilities, SHSP, foundation match |

## Grant narrative angle

> Rural recreation corridors around Lake Havasu and Mohave County create high-risk search-and-rescue conditions: extreme heat, desert terrain, boating incidents, OHV crashes, remote trails, limited cell service, and long response times. A UAS readiness program gives responders thermal search, rapid aerial assessment, search-grid mapping, safer reconnaissance, and faster incident command visibility.
