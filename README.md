# Rural Response Drones Docs

Research and planning docs for a rural search-and-rescue / public-safety drone venture focused on Lake Havasu, Mohave County, and similar rural recreation markets.

## Stack

- Astro
- Starlight
- Vercel

## Development

```bash
npm install
npm run dev
npm run build
```

## Deployment

```bash
HOME=/home/admin vercel --prod --yes
```

## Privacy posture

This is an internal planning site. It includes `robots: noindex,nofollow` until claims and partner-sensitive details are approved for public use.
